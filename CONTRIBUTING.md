# Contributing

**We are not accepting pull requests at this time.** The codebase is still evolving quickly and we can't yet commit to the review bandwidth that external contributions deserve. This will change — watch this file.

What we *do* welcome right now:

## Bug reports

Found something broken? [Open a bug report issue](https://github.com/general-intelligence-inc/tensorchat/issues/new?template=bug_report.md). Please include:

- Your device model and OS version
- The app version (shown in Settings)
- Which GGUF model + quantization you were running
- Steps to reproduce
- What you expected vs. what happened
- Any relevant logs from the console

The more specific, the faster we can fix it.

## Feature requests

Have an idea? [Open a feature request issue](https://github.com/general-intelligence-inc/tensorchat/issues/new?template=feature_request.md). Tell us the problem you're trying to solve first — the solution comes second.

Things that are **out of scope** and will be declined:

- Cloud LLM inference, telemetry, analytics, or user accounts — TensorChat is on-device by design
- Opt-out network features — anything that talks to the network must be opt-in and off by default

## Security issues

**Do not file security vulnerabilities as public issues.** See [SECURITY.md](SECURITY.md) for private disclosure.
